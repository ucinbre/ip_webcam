üìñ Panduan Lengkap: Scrcpy (Solusi Terbaik)

Windows:

  1. Download scrcpy dari: https://github.com/Genymobile/scrcpy/releases
  2. Extract file ZIP
  3. HP: Settings ‚Üí About Phone ‚Üí Tap "Build Number" 7x
  4. Settings ‚Üí Developer Options ‚Üí USB Debugging (ON)
  5. Hubungkan HP ke laptop via USB
  6. Jalankan scrcpy.exe

Mac:

  1. Install Homebrew jika belum: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install scrcpy: brew install scrcpy
  3. Setup USB Debugging di HP (sama seperti di atas)
  4. Hubungkan HP via USB
  5. Terminal: scrcpy

Linux:

  1. Ubuntu/Debian: sudo apt install scrcpy
  2. Arch: sudo pacman -S scrcpy
  3. Setup USB Debugging di HP
  4. Hubungkan USB
  5. Terminal: scrcpy

üì± Untuk Wireless (setelah setup USB pertama kali):

  1. Hubungkan USB dulu, jalankan: scrcpy --tcpip
  2. Cek IP HP di Settings ‚Üí About ‚Üí Status
  3. Cabut USB
  4. Jalankan: scrcpy --tcpip=192.168.x.x:5555 (ganti dengan IP HP)
